# Interdisciplinary Livestock Microbiome Research Lab (ILMRL) Website

This repository hosts the public website for the ILMRL.

## Structure
- `index.html`: The main homepage.
- `css/style.css`: All styling and colors.
- `images/`: Stores the logo and asset images.

## Updating the Site
To add a new publication or news item, edit `index.html` and look for the "News" or "Publications" section.
